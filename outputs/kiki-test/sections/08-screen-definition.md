# 화면 정의 (Screen Definition)

본 문서는 KIKI Shop 서비스의 화면별 상세 정의를 기술한다.

---

| 화면 ID | 화면명 | 목적 | 프로세스 단계 |
|---------|--------|------|--------------|
| SCR-001 | 메인 화면 | 서비스 진입점, 주요 상품 노출 | 진입 |
| SCR-002 | 로그인/회원가입 | 사용자 인증 | 인증 |
| SCR-003 | 상품 목록 | 카테고리별 상품 브라우징 | 탐색 |
| SCR-004 | 상품 상세 | 개별 상품 정보 확인 | 탐색 |
| SCR-005 | 장바구니 | 선택 상품 관리 | 주문준비 |
| SCR-006 | 주문/결제 | 배송지 및 결제 | 결제 |
| SCR-007 | 주문 완료 | 주문 성공 확인 | 완료 |

## SCR-001: 메인 화면

### 기본 정보
- **화면 ID**: SCR-001
- **화면명**: 메인 화면
- **목적**: 서비스 진입점으로서 주요 상품, 프로모션, 카테고리 노출
- **URL**: `/`
- **진입 조건**: 없음 (누구나 접근 가능)
- **이탈 조건**: 다른 화면으로 이동

### UI 구성 요소

| 요소 ID | 요소 유형 | 설명 | 표시 조건 |
|---------|----------|------|----------|
| NAV-001 | 네비게이션 | 상단 GNB (로고, 검색, 장바구니, 마이페이지) | 항상 |
| CARD-001 | 카드 | 메인 배너 슬라이더 | 항상 |
| LIST-001 | 리스트 | 추천 상품 목록 | 항상 |
| LIST-002 | 리스트 | 베스트 상품 목록 | 항상 |
| BTN-001 | 버튼 | 카테고리 바로가기 | 항상 |

### 기능 정의

| 기능 | 사용자 액션 | 시스템 처리 | 결과 화면 |
|------|-----------|------------|----------|
| 검색 | 검색어 입력 후 엔터 | 검색 API 호출 | SCR-010 |
| 상품 클릭 | 상품 카드 클릭 | - | SCR-004 |
| 장바구니 | 장바구니 아이콘 클릭 | - | SCR-005 |
| 로그인 | 로그인 버튼 클릭 | - | SCR-002 |

## SCR-002: 로그인/회원가입

### 기본 정보
- **화면 ID**: SCR-002
- **화면명**: 로그인/회원가입
- **목적**: 사용자 인증 및 신규 회원 등록
- **URL**: `/auth`
- **진입 조건**: 비로그인 상태
- **이탈 조건**: 로그인 성공 또는 취소

### UI 구성 요소

| 요소 ID | 요소 유형 | 설명 | 표시 조건 |
|---------|----------|------|----------|
| FORM-001 | 폼 | 로그인 폼 | 로그인 탭 선택 시 |
| FORM-002 | 폼 | 회원가입 폼 | 회원가입 탭 선택 시 |
| INPUT-001 | 입력 필드 | 이메일 입력 | 항상 |
| INPUT-002 | 입력 필드 | 비밀번호 입력 | 항상 |
| BTN-002 | 버튼 | 로그인 버튼 | 로그인 탭 |
| BTN-003 | 버튼 | 회원가입 버튼 | 회원가입 탭 |
| BTN-004 | 버튼 | 소셜 로그인 (카카오, 네이버) | 항상 |

### 기능 정의

| 기능 | 사용자 액션 | 시스템 처리 | 성공 시 | 실패 시 |
|------|-----------|------------|---------|---------|
| 로그인 | BTN-002 클릭 | POL-AUTH-001 적용, 토큰 발급 | 이전 화면 복귀 | 에러 메시지 표시 |
| 회원가입 | BTN-003 클릭 | POL-AUTH-002 검증, 계정 생성 | 로그인 탭 이동 | 에러 메시지 표시 |
| 소셜 로그인 | BTN-004 클릭 | OAuth 인증 | 메인으로 이동 | 에러 메시지 표시 |

### 관련 정책
- POL-AUTH-001: 로그인 실패 제한
- POL-AUTH-002: 비밀번호 정책
- POL-VAL-001: 이메일 형식 검증

# 용어 정의 (Glossary)

본 문서에서 사용되는 주요 용어를 정의한다.

---

## 비즈니스 용어

| 용어 | 정의 | 사용 맥락 |
|------|------|----------|
| SKU | Stock Keeping Unit. 재고 관리를 위한 상품 고유 식별 단위 | 상품 등록, 재고 관리 |
| PG | Payment Gateway. 온라인 결제를 중개하는 서비스 | 결제 처리 |
| CS | Customer Service. 고객 문의 및 불만 처리 서비스 | 주문 취소, 환불 |
| GMV | Gross Merchandise Volume. 총 거래액 | 매출 리포트 |
| AOV | Average Order Value. 평균 주문 금액 | 매출 분석 |

---

## 주문/결제 용어

| 용어 | 정의 | 사용 맥락 |
|------|------|----------|
| 주문번호 | 주문 생성 시 부여되는 고유 식별자 (형식: ORD-YYYYMMDD-XXXXX) | 주문 조회, CS 처리 |
| 결제대기 | 주문은 생성되었으나 결제가 완료되지 않은 상태 | 가상계좌, 무통장입금 |
| 배송준비중 | 결제 완료 후 상품 출고 전 상태 | 주문 상태 |
| 배송중 | 택배사에 상품이 인계된 상태 | 주문 상태 |
| 배송완료 | 수령인에게 상품이 전달된 상태 | 주문 상태 |

---

## 회원 용어

| 용어 | 정의 | 사용 맥락 |
|------|------|----------|
| 게스트 | 비로그인 상태의 사용자 | 비회원 주문 |
| 회원등급 | 구매 실적에 따라 부여되는 등급 (일반/VIP/VVIP) | 할인율 적용 |
| 적립금 | 구매 시 적립되어 다음 주문에 사용 가능한 포인트 | 결제 |
| 쿠폰 | 특정 조건에서 할인을 제공하는 코드 | 프로모션 |

---

## 상품 용어

| 용어 | 정의 | 사용 맥락 |
|------|------|----------|
| 옵션 | 동일 상품 내 선택 가능한 변형 (색상, 사이즈 등) | 상품 상세 |
| 품절 | 재고가 0인 상태 | 상품 목록, 장바구니 |
| 일시품절 | 일시적으로 재고가 없으나 재입고 예정인 상태 | 상품 상세 |
| 판매종료 | 더 이상 판매하지 않는 상품 | 상품 관리 |

---

## 기술 용어

| 용어 | 정의 | 사용 맥락 |
|------|------|----------|
| API | Application Programming Interface. 시스템 간 통신 인터페이스 | 개발 |
| 토큰 | 사용자 인증 정보를 담은 암호화된 문자열 | 로그인 유지 |
| 세션 | 사용자의 연결 상태를 유지하는 서버 측 저장소 | 장바구니 |
